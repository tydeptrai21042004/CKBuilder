# Community interoperability assets

This directory contains artifacts intended for reuse outside the capstone application.

- `decoder/credential-cell-decoder.js`: dependency-free reference decoder.
- `test-vectors/credential-cell-v1.json`: deterministic conformance corpus.
- `schemas/`: JSON Schemas for the vector corpus and exported public proof.
- `decoder/decoder-manifest.example.json`: example metadata a CKB explorer or visualizer can use to recognise a deployment before applying the decoder.

These artifacts describe a community protocol, not an official Nervos standard.
