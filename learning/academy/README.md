# CKB Academy evidence

Create one file per completed module. Do not copy generated summaries as proof of completion.

Each file should record:

- module title and URL;
- date completed;
- score or completion result;
- three concepts explained in your own words;
- one question or difficulty;
- screenshot path;
- how the concept was used in this repository.

Suggested names: `module-01.md` through `module-08.md`.
