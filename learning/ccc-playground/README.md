# CCC Playground evidence

For each exercise, record:

1. exercise name;
2. code written personally;
3. network used;
4. command or Playground steps;
5. transaction hash or result;
6. screenshot;
7. explanation of inputs, outputs, Lock Script, Type Script, and Cell data where applicable.

Recommended first evidence:

- `transfer-ckb.md`
- `store-data-on-cell.md`

Using CCC inside the capstone does not by itself prove that the Playground exercise was completed.
