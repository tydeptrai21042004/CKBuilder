# CKBuilder handbook progress tracker

This file separates programme evidence from capstone engineering. A code feature does not automatically prove completion of a handbook exercise.

## Reporting rules

- Publish one report each week on the chosen day.
- Describe only work completed during that reporting period.
- Do not upload several retrospective reports together.
- Include commands, screenshots, scores/results, blockers, and personal learning.
- AI may help tidy writing, but the report must reflect the participant's actual work and understanding.
